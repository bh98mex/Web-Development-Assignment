A collection of functional webpageds built in HTML and PHP.
Contains:
- Currency Converter;
- Registration Form;
- Weathr information retrieval based on a informations aquired from the internet;
- Youtube Data API integration - create your own videos viewing page based on the YouTube data API;
- Guardina news API - get the latest news from Guardina, based on their application programming interface.

The pages will only function with Edge explorer and need a database (included), Php MyAdmin, and the Carbon library 
for the Guardian API.
