A collection of HTML/PHP pages.
Contains:
- Currency Converter;
- Registration Form;
- Weather information retrieval, based on informations aquired from the internet;
- Youtube Data API integration - create your own videos viewing page, based on the YouTube data API;
- Guardina news API - get the latest news from Guardian, based on their application programming interface.

The pages will only function with Edge explorer. A database is required (included), Php MyAdmin, and the Carbon library 
for the Guardian API.
